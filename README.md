 # 一、基础
 ## 1.分类（根据训练数据是否有标记信息）
（1）、监督学习（划分==*回归/分类*==）： 学习输入到输出orX到Y（true/false,无限种某种结果）的映射。
	A、回归：从无数多的正确值学习，可能输出数字中（一个）预测数值（连续值）。
	B、分类：预测的结果有少数的可能/输出，预测类别，结果不一定是数字（离散值）
（2）、无监督学习（==聚类==，还有异常检测与降维）：不提供正确（有标签）的答案，让算法自己去发现数据的类别，并放到不同的簇（组）中。

 ## 2.线性回归模型
训练模型：
	训练集：包括输入特征和输出目标 ->输入到训练算法f中
  ## 3.成本(假设)函数(==线性回归模型==)
$$
  f_{w,b}(x)=wx+b=y
$$
	
其中w,b是系数或者权重，可以==调整w,b==来优化模型,目的就是寻找最优的w,b。
x即线性回归模型的==特征==。

测量误差：(y*-y)^2,再相加除以m -> 平均误差平方和,就得到了
==成本函数:==
	
$$
	J(w, b) = \frac{1}{2m} \sum_{i=1}^{m} (f_{w,b}(x^{(i)}) - y^{(i)})^2
$$
	想要得到minJ(w,b)
4.线性回归的目标就是：**找到一组最优的 w 和 b** ，使得模型的预测值 fw,b(x)尽可能接近真实标签 y。通常通过**最小化损失函数**
